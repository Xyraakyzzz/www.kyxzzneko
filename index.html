Integrasi Super Mulus

Integrasikan Nanzz API ke dalam aplikasi Anda hanya dengan beberapa baris kode. Kami mendukung berbagai bahasa dan framework untuk pengalaman yang lancar.

1 Instal paket melalui NPM

npm install api-nanzz

2 Gunakan dalam proyek Anda

Inisialisasi dan panggil endpoint dengan mudah menggunakan async/await.

Baca Dokumentasi
app.js 
import nanzz from 'api-nanzz';

async function fetchData() {
  /* Mengambil data dari endpoint kategori AI */
  const response = await nanzz.get('/category/ai', {
    example: 'Testing',
    limit: 5,
    format: 'json'
  });

  console.log(response.data);
}

fetchData();

Temukan jawaban untuk pertanyaan mengenai layanan kami.

Apa itu Nanzz API?
Nanzz API adalah platform yang menyediakan berbagai macam antarmuka pemrograman aplikasi (API) siap pakai untuk membantu developer membangun aplikasi web, seluler, dan bot otomatisasi dengan lebih cepat tanpa harus membangun backend dari nol.
Apakah layanan ini benar-benar gratis?
Ya, paket "Gratis Selamanya" kami menawarkan permintaan tanpa batas ke semua endpoint publik kami tanpa memerlukan kartu kredit atau komitmen berlangganan. Kami percaya dalam mendukung komunitas developer.
Bahasa pemrograman apa yang didukung?
Karena kami menyediakan REST API standar yang mengembalikan respon JSON, Anda dapat menggunakan layanan kami dengan hampir semua bahasa pemrograman, termasuk JavaScript (Node.js, React), Python, PHP, Java, Go, Ruby, dan banyak lagi.
